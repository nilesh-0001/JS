In this session we are gonna talk about Difference btw let const and var
  1. Let

    2. It can be updated but not re-declared

  2. var

    3. var is Globally scoped and let or const are block scoped

    var can be redeclared and update within its scope

  3. const

    4. const are neither be updated or nor be re-declared

var variables are initalized with undefined where as let and const are not initalized
const must be initialized during declaration unlike let and var
